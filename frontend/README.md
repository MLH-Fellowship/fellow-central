# Fellow Central - Frontend

Frontend web app of Fellow Central.

## Technologies Used

- React
- Redux
- Sass
- Iconify

## Setup Dev Enviroment

1. Install [Node.js](https://nodejs.org/en/)

2. Install project dependencies by running the command from `frontend/` directory:
    ```bash
    npm install
    ```

3. Start dev server:
    ```bash
    npm run start
    ```
